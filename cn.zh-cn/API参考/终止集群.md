# 终止集群<a name="ZH-CN_TOPIC_0109962551"></a>

## 功能介绍<a name="section4408504619327"></a>

数据完成处理分析后或者集群运行异常无法提供服务时可终止集群服务。该接口兼容Sahara。

处于如下状态的集群不允许终止：

-   scaling-out
-   scaling-in
-   starting
-   terminating
-   terminated
-   failed
-   scaling-error
